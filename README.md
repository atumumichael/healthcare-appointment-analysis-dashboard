# Healthcare Appointment Analysis Dashboard

## 📊 Overview
This project analyzes healthcare appointment data to understand patient attendance, waiting time, and demographic trends.

## 🎯 Objective
To reduce missed appointments and improve hospital efficiency.

## 🛠 Tools Used
- Power BI
- Excel

## 📈 Key Metrics
- Total Appointments: 39K
- Missed Appointments: 8K
- Patients Attended: 31K
- Avg Waiting Time: 10.88 days

## 🔍 Key Insights
- High number of missed appointments (~20%)
- Majority of appointments had short waiting times
- Some neighborhoods had higher appointment volume
